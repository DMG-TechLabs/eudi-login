# eudi-login

## Get Started

```bash
$ git clone https://github.com/DMG-TechLabs/eudi-login
$ cd eudi-login

$ make start
```

## Endpoints

- localhost:80: Demo App using the `eudi-login`
- localhost:8080: The `eudi-login` service
